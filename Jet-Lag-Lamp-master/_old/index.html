Vision statement

Jet lag is a common problem for many travelers and for many workers in the world. This AmI can help to ease the change of circadian rhythm.

The systeme will be used in hotel's rooms. It will help the customer of the hotel room to find the best time to sleep in order to minimize the effects of the jet lag. Here we propose a solution which enables the user to sleep at a precise time and in the best conditions.

AMI main steps

Ami step Description
Sensing The ami system is capable to use the data in the smartphone (google agenda,...) to know what are the obligations of the user. Some sensors (temperature and brightness) will be used to know when open/close the blinds or switch on/off the lights.
Reasoning The Ami system can calculate the circadian rythm of the user with some informations about his sleeping rythm. It can also read (receive) the user's appointments and give advices to the user in using the data collected from the smartphone.
Acting The Ami system is capable to switch on/off the lights, open/close the blinds, give advice to the user with a microphone, a telegram bot or a notification.
Interacting The system can give advices to the user for his schedule, suggest some naps to the user during the day, ask feedback to the user.

AMI features

Ami step Description
Sensitive The Ami system can know if the user is inside the hotel room or not. If he is inside, it can collect data from his smartphone to advice him.
Reasponsive If the system calculate that the user needs to sleep, it will advice him to take a break. The user can refuse but if he accepts, the system will calculate the time that he needs to sleep and fix the alarm. It will also close the blinds and switch off the lights.
Adaptive The system will ask to the user some informations about his sleep to calculate the best way how long he needs to sleep and when.
Transparent
Ubiquitous
Intelligent Ce system will be intelligent because it will store the data of every users that will come to the hotel room into a database. If a user is used to come in a this hotel, we do not have to ask him again all the information

Open issues

To suceed in doing this project, we will need to get access to the smartphone of the user especially we will use the google agenda and the alarm menu. After that, we need to calculate, thanks to the data given by the user, the best way for him to sleep and to accustom himself to the circadian rhythm of the region. We will need to use bluetooth or wifi technologie to control the lamps of the room and the blinders. We will also use a telegram bot, notifications of the phone or a microphone to interract with the user.
