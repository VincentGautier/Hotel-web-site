Sensing

The ami system is capable to use the data in the smartphone (google agenda,...) to know what are the obligations of the user. Some sensors (temperature and brightness) will be used to know when open/close the blinds or switch on/off the lights.

Reasoning

The Ami system can calculate the circadian rythm of the user with some informations about his sleeping rythm. It can also read (receive) the user's appointments and give advices to the user in using the data collected from the smartphone.

Acting

The Ami system is capable to switch on/off the lights, open/close the blinds, give advice to the user with a microphone, a telegram bot or a notification.

Interacting

The system can give advices to the user for his schedule, suggest some naps to the user during the day, ask feedback to the user.
