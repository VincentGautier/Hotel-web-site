Vision

Jet lag is a common problem for many travelers and workers all around the world.
The Jet Lag Lamp can ease the change of circadian rhythm.

The system will be used in hotel's rooms. To be more precise, this ami could be used for a particular hotel chain. Like this, we can use all the information given by the user for an other trip. It will help the customer of the hotel room to find the best time to sleep in order to minimize the effects of the jet lag. Here we propose a solution which enables the user to sleep at a precise time and in the best conditions. We will use all the environnement given by the hotel chain to personalize and optimise the sleep of the user. To permite to the user to sleep in the best conditions, we wil use sensors and science. Some feedbacks will be ask to the user to make sure of his satisfaction.
