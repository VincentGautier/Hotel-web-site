Our rooms

Why our project is intelligent.

Sensitive

The Ami system can know if the user is inside the hotel room or not thanks the wifi. If he is inside, according to the data inside the google calendar, we can advice him to sleep and if he agrees, we prepare the room for his sleep ( shutting the blinds, switch off the lights,...)

Responsive

If the system calculate that the user needs to sleep, it will advice him to take a break. The user can refuse but if he accepts, the system will calculate the time that he needs to sleep and fix the alarm. It will also close the blinds and switch off the lights.

Adaptive

The system will ask to the user some informations about his age. Using the science, we will claculate the best time of sleeping. Moreover, using the google calendar, we know when he has appointments or other flights, we can anticipate all of this and advice him to sleep at a preise time.

Intelligent

Ce system will be intelligent because thanks the science and the age, it can calculates the sleeping time of the user. It can also integrate to his calculation the feedback given by the user. Like this it permites to personalize the sleeping time.
